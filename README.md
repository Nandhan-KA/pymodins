# PYMODINS

pymodins is a Python Modules Installer for Developers and Peoples who are new to python.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install pymodins.

```bash
pip install pymodins
```

## Usage

```python
import pymodins

# runs
pymodins.run()

```
